# 「BlogOne」/ 「一个 · 轻博客」
>A Go Blog. Simple and Powerful. 
---
### 说明文档

「一个 · 轻博客」（BlogOne）是一个GO语言编写的博客构建工具，可以快速搭建博客网站。它无依赖跨平台，配置简单构建快速，注重简洁易用与更优雅的排版。
### 灵感
这是受ONE 启发所想到的，开始只是一个界面。现在想把想法全部实现出来。
希望能吸收微博和博客的一些优点，在这个程序上实现一些我自己的想法，我想它是个有个性的博客程序。
